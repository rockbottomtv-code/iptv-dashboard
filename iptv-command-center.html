How XUI Sync works: Enter your panel URL + admin credentials below. The dashboard calls the XUI API to pull all lines and their expiration dates. You then link each line to a customer. Linked lines stay remembered via localStorage. 8K Gold (my8k.me) and SuperAdmin don't have public APIs — use Quick Open to access them directly.
🏠 Home Assistant Integration

Host this HTML file on your local server (Raspberry Pi, NAS, always-on PC), then add a Webpage Card in your HA dashboard:

type: iframe
url: http://YOUR-LOCAL-IP:8080/iptv-command-center.html
aspect_ratio: 100%

To serve the file: python3 -m http.server 8080 in the same folder.
All your credentials and customer data stay saved in the browser localStorage on that device.
